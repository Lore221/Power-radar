# Power Radar Ponder Scene Plan

## Planned category

Radar Systems

## Planned scenes

1. Building a Fixed Radar

   * Radar Controller
   * Fixed Antenna Mount
   * Antenna Array Core
   * Radar Panels
   * panel placement assist

2. Connecting a Remote Monitor

   * Radar Link
   * tuning held Link
   * blue outline
   * max distance 128
   * Radar Monitor Controller
   * first Radar Display placed directly on Controller
   * display wall expansion
   * valid square sizes 1x1 through 5x5
   * inner frame disappearance
   * GUI via active Display
   * invalid multiple-controller conflict
   * remote monitor

3. Radar Scan Modes

   * ground scan
   * sky scan
   * FOV sector
   * monitor display

4. Invalid Radar Network

   * multiple radars
   * out of range
   * controller offline

5. Future Rotary Antenna Mount

   * Create kinetics
   * visual rotation
   * 360-degree scan

6. Displaying Track Count

   * Create Display Link
   * Radar Controller as a display source
   * detected tracks count
   * Display Board / Nixie Tubes

Real scenes are intentionally deferred to a separate implementation stage.
Schematic structures, storyboard timings, custom Ponder tags, and user-visible Ponder entries are not created yet.
